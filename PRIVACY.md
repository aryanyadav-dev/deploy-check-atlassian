# Privacy Policy

**Last updated:** December 17, 2025

## Overview

ShipGuard (Deployment Risk Analyzer) is an open-source tool that analyzes code changes for deployment risks. This privacy policy explains how the application handles data.

## Data Collection

ShipGuard does **not** collect, store, or transmit any personal data to external servers.

### What the app accesses:
- **Jira Issues**: Read issue details to display deployment risk analysis
- **Confluence Pages**: Read/write pages to publish risk reports
- **Bitbucket PRs**: Read PR details for automated analysis (when configured)
- **App Storage**: Stores configuration settings within your Atlassian instance

### What the app does NOT do:
- Does not collect personal information
- Does not track user behavior
- Does not share data with third parties
- Does not store data outside your Atlassian instance

## Data Storage

All data is stored within your Atlassian Cloud instance using Atlassian's secure storage APIs. No data leaves the Atlassian ecosystem.
